# AmigoPet — Pacote de Identidade Visual

Arquivos inclusos:

- assets/amigopet-logo-horizontal.svg
- assets/amigopet-icon.svg
- assets/favicon.svg
- assets/amigopet-brand.css
- assets/brand-tokens.json
- guia_identidade_visual.html
- landing_amigopet.html
- texto_comercial_amigopet.md

Como usar:
1. Copie os arquivos de `assets/` para `frontend/assets/`.
2. Use `favicon.svg` no `<head>`.
3. Use `amigopet-icon.svg` como base do ícone Android.
4. Use `landing_amigopet.html` como página comercial.
